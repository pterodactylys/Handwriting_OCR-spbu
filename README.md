## Распознавание рукописного текста с анализом эмоциональной окраски 


## Описание проекта

Цель — разработать программное обеспечение, которое:
1. Принимает на вход изображение с рукописным текстом на русском или английском языке.
2. Преобразует его в цифровой текст с помощью модели глубокого обучения (CRNN + CTC).
3. Анализирует эмоциональную окраску (тональность) распознанного текста, используя лексиконный подход.

Основной фокус проекта — реализация алгоритмов распознавания рукописного текста. Анализ эмоциональной окраски - побочная задача.

## Возможности (примерный план, будет корректироваться в процессе работы)

- **Предобработка изображений** – бинаризация, выравнивание строк, изменение размера, нормализация.
- **Распознавание рукописного текста** – свёрточно-рекуррентная нейронная сеть (CRNN) с CTC‑loss.
- **Анализ тональности** – лексиконный метод на основе готовых словарей.
- **Веб‑интерфейс** для загрузки изображений и просмотра результатов.

## Датасет

- **Английский** Для обучения и оценки будет использована база данных [IAM Handwriting Database](http://www.fki.inf.unibe.ch/databases/iam-handwriting-database). Она содержит изображения рукописных текстов на английском языке с постраничной и построчной разметкой (транскрипциями).

- **Русский** Для обучения будет использована база данных [HTR Dataset](https://link.springer.com/article/10.1007/s11042-021-11399-6).

1. Клонируйте репозиторий:
   ```bash

2. Создайте виртуальное окружение (рекомендуется):
